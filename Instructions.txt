In order to use this addin, you must ensure that 'qwen2:1.5b-instruct-q4_K_M' is installed via Ollama (https://ollama.com/). You can change the model used by this addin at any time by changing a user environment variable called 'EXCEL_MODEL'. Place the addin in an appropriate location, such as '%APPDATA%\Microsoft\AddIns'. Next, double click on the addin and select 'trust this publisher'. After that, make sure to follow these instructions:
=> Add Options -> Add-ins -> (Manage: Excel Add-ins Go) -> Browse -> Click on Ask-LLM-AddIn64-packed.xll -> Make sure the checkbox is selected on Excel.
